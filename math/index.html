Useful Math

Here is a brief reference of math that you thought you'd never use again in your life. WARNING: Below you will find calculus and trigonomitry and you may end up liking them at the end of this. You've been warned.

The Basic Four

Adding

  • Obvious: var c = a + b;
  • Short Assignment: b += a; increases b by a and saves it.
  • Incrementing: You can use n++; to add 1 to a number.
    • Specifically, it'll use the number first, then add 1. ++n; will add 1 THEN use.
    • So var a = 10; point(0, a++); will draw a point at (0, 10) but point(0, ++a); will put it at 11.
  • Strings: "a" + "b" = "ab", "a" + 9 = "a9"
  • Arrays behave like strings at best: 8 + [9,3] = "89,3", "a" + [9,3] = "a9,3"

Subtracting

  • Obvious: var c = a - b;
  • Short Assignment: b -= a;

Multiplying

  • Obvious: var c = a * b;
  • Short Assignment: b *= a;
  • Powers: pow(a, 2) is a squared

Dividing

  • Obvious: var c = a / b;
  • Short Assignment: b /= a;
  • Remainders with % (modulo): 10 % 4 = 2. Divides and returns the remainder.
  • Modulo Assignment: b %= a; if b = 13 and a = 3, b %= a; sets b to 1.
  • Odd and Even: num % 2 will return 1 if the number is odd and 0 if it's even.

Radians

A brief reminder: the default angleMode for Processing projects is radians, which go from 0 to 2 π. Processing has some built in constants to help with this, namely PI, HALF_PI, and TWO_PI. However, even if you draw with angles, other trig functions (below) will still expect radians. Converting from degrees to radians looks like this: rad = deg*PI/180 and from radians to degrees deg = rad*180/PI.

Angles to Coordinates

Remember sine and cosine? SUPER useful when it comes to placing graphics and creating smooth motion. In geometry, cos() will give you the percent of the magnitude that is in the x direction, and sin() will give you the y. In Processing, this would put 0 radians/degrees pointing straight right.

A trick, instead of subtracting 90deg from the angle is to use + sin() for x and - cos() for y.

var aslider;

function setup() {
  createCanvas(300, 300);

  aslider = createSlider(0, TWO_PI*100, PI*100);
  aslider.position(0, 310);
  aslider.size(width);
}

function draw() {
  background(200);

  fill(0);
  strokeWeight(1);
  var frameAngle = frameCount*PI/180;
  ellipse(sin(frameAngle)*(width/2-50)+width/2, 25, 25 ,25);
  ellipse(25, cos(frameAngle)*(height/2-50)+height/2, 25 ,25);
  ellipse(width/2+sin(frameAngle)*111, height/2+cos(frameAngle)*111, 10, 10);

  fill(255);
  strokeWeight(3);
  ellipse(width/2, height/2, 200, 200);
  stroke(0);
  var angle = aslider.value()/100;
  line(
    width/2  + sin(angle)*50,
    height/2 - cos(angle)*50,
    width/2  + sin(angle)*100,
    height/2 - cos(angle)*100
  );
}

Calculating Angles

One common math problem in visual programming is determining the angle between two points. Luckily, arctangents exist, and they're this easy:

function setup() {
  // set canvas size
  createCanvas(200, 200);
  noStroke();
}

function draw() {
  // Clear
  background(50);

  // CALCULATE ANGLE FROM CENTER TO MOUSE
  var angle = atan2(mouseY-height/2, mouseX-width/2);

  // Eye
  fill(255);
  ellipse(width/2, height/2, 100, 100);
  fill(0, 0, 150);
  ellipse(width/2+cos(angle)*25, height/2+sin(angle)*25, 50, 50);
}
