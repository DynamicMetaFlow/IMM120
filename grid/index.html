Moving and Shaking

The Drawing Grid

Everything you draw in a sketch is drawn onto a coordinate system, very similar to a graph in regular math, only flipped vertically.

Transforming the Grid

The Processing language gives you the unique power to move and rotate this grid as you please, using translate and rotate. This can be very useful when you want to spin parts of your sketch around.

Remember: the computer will still draw relative to the origin. translate(10, 10); ellipse(10, 10, 5, 5); will draw a tiny circle centered at 20, 20.

function setup() {
  // set canvas size
  createCanvas(200, 200);
}

function draw() {

  // Move the grid 200px right and 10px down
  translate(200, 10);
  // Rotate around origin, you can use RADIANS or DEGREES, choose with angleMode();
  rotate(HALF_PI);

  // DRAW

}

Drawing with Multiple Transformations

You are not limited to one transformation per draw loop. You can store your current transformations to the grid using push() to return to earlier states with pop() later. Using the stack to its full potential is often necessary to create the visuals you desire in a sketch.

function setup() {
  var arrowCanvas = createCanvas(300, 300);
  arrowCanvas.parent('spin');
  angleMode(DEGREES);
}

function draw() {
  background(255);

  // Red rectangle
  // Translate, then rotate
  fill(255, 0, 0);
  push(); // Save translate(0, 0) and rotate(0)
    translate(150, 75);
    rotate(frameCount);
    line(-100, 0, 100, 0);
    line(0, -100, 0, 100);
    rect(0, 0, 60, 80);
  pop(); // Back to 0,0:0

  // Green rectangle
  // Rotate, then translate
  fill(0, 255, 0);
  push();
  rotate(frameCount);
  translate(150, 150);
  line(-100, 0, 100, 0);
  line(0, -100, 0, 100);
  rect(0, 0, 60, 80);
  pop();

  // Blue rectangle
  // Translate, then rotate, draw at a position of -w/2, -h/2
  //   where w and h are the width and height of the the rectangle
  fill(0, 0, 255);
  push();
  translate(150, 225);
  rotate(frameCount);
  line(-100, 0, 100, 0);
  line(0, -100, 0, 100);
  rect(-30, -40, 60, 80);
  pop();
}
