Functions

or the benefits of organizing code

function (noun)

a named block of code

block (noun)

a group of code, normally separated from the rest of the code by { being between braces }

block (cont)

variables created in a block cannot be used outside the block, but can be used in blocks inside said block

// Example
var a = 3;

if (question) { // Block 1
    var sum = 0;
    sum = sum + a;

    // Block 2
    if (q2) {
        var b = 7;
        sum = sum + b; // sum is defined "above", so we're good
    } // end Block 2

    println(sum); // sum created inside this block, good
    println(b);   // b is lost at the end of Block 2
} // end Block 1

println(sum); // error, sum was created in Block 1

Functions are excellent for code that does one particular thing.

Especially if you want to do that thing multiple times.

// Draw an apple where the mouse is
function drawApple() {
  // Stem
  noStroke();
  fill('green');
  rect(mouseX-4, mouseY-60, 8, 20);
  // Flesh
  fill('red');
  stroke('maroon');
  ellipse(mouseX, mouseY, 100, 100);
}
            
function setup() {
    createCanvas(640, 480);
    background('blue');
}

function draw() {
    // No background
    if (isMousePressed) {
        drawApple();
    }
}
