Getting Started

  • Download an empty sketch
  • Find and unzip the file (Mac: double-click, Windows: right-click > Extract All)
  • Rename the new_sketch folder
  • In Brackets, File > Open Folder
  • Select the folder and click the Open button
  • On the left, double-click on sketch.js and hit the lightning bolt (top right) to run

Making A New Sketch (New File Method)

  • File > New File
  • Save it in the same folder as NAME.js
  • On line 8 of index.html, change sketch.js to NAME.js
  • Refresh or relaunch preview

Making A New Sketch (Folder Method)

  • Go back to the zip file (likely in Downloads)
  • Extract the zip, will create a new folder
  • Rename the folder
  • Open Folder in Bracket
  • If you want another Brackets window, go to Debug > New Window

Useful Code

function isMouseInsideRect(x, y, w, h) {
    return mouseX > x && mouseY > y && mouseX < x+w && mouseY < y+h;
}
